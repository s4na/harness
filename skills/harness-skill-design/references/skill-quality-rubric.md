# Skill Quality Rubric

Score each skill from 0 to 2 in each category.

- Trigger quality: clear when to use, includes likely user vocabulary, avoids overbroad activation.
- Actionability: gives concrete steps, examples, and expected outputs.
- Composability: names related skills and avoids duplicating unrelated domains.
- Traceability: links to ADRs or explains why no ADR is needed.
- Enforceability: identifies checks that can make important guidance deterministic.

A new skill should score at least 8/10 before merging.
