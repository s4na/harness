# Repository Agent Initializer

This repository is an AI development harness. When working here, first load the repository-development skills that define how the harness should evolve:

1. `skills/core-adr/SKILL.md` for new or changed design decisions.
2. `skills/core-code-review/SKILL.md` before reviewing any change.
3. `skills/harness-skill-design/SKILL.md` before creating or editing skills.
4. `skills/harness-repository-maintenance/SKILL.md` before changing repository structure, manifests, configs, or adapters.
5. `skills/llm-prompt-engineering/SKILL.md` when writing prompts, skill descriptions, agent instructions, or evaluation prompts.

Keep this file intentionally small. It is an initializer that points agents to the repository-specific skills rather than duplicating their guidance.
